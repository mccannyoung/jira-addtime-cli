# jira-addtime-cli
A simple go program to add time to a JIRA issue via commandline.

use:
```
\main.exe:
  -ticket string
        The ticket to add time to - please use the format of PROJ-ticketnumber
  -time string
        Please user Jira's time format of 1h 30m to log time
  -v    Shows version
  -worklog string
        The worklog comment
exit status 2
```
